Add Bootstrap Elements to your pages using Visual Editors.
Tiny Bootstrap Elements Light adds a Bootstrap Toolbar to TinyMce Rich Text Editor.

== Description ==

Tiny Bootstrap Elements Light is the perfect tool for WordPress to add Bootstrap elements to your pages.

Add a **Bootstrap toolbar** to WordPress editor, access to visual editors to create/edit :
* Buttons (+ icons)
* Icons                       [PRO version only]
* Tables                      [PRO version only]
* Templates (create only)     [PRO version only]
* Breadcrumbs                 [PRO version only]
* Paginations                 [PRO version only]
* Pagers                      [PRO version only]
* Labels
* Badges
* Alerts
* Panels                      [PRO version only]

Can be used with **any WordPress theme** : if your theme is based on Bootstrap, just check option to use your theme in WordPress editor.
Otherwise, you can add default or custom Bootstrap css, including only needed elements.

Each element on Tiny Bootstrap Elements toolbar can be toggled individually.

Each element is customizable with usual Bootstrap options. Total Font Awesome support.

**Use your own Bootstrap css**, wich will be directly rendered either in your public pages, WordPress editor and plugin visual editors.

 * 12 plugins in 1
 * Visual Editors
 * Lightweight plugin, adds only the needed css on pageload
 * Easy to install, no skill needed

**PRO version** available at <a href="http://codecanyon.net/item/tiny-bootstrap-elements-wordpress-plugin/10293837" target="_new"><u>http://codecanyon.net/item/tiny-bootstrap-elements-wordpress-plugin/10293837</u></a>


== Installation ==

    Unzip
    Copy tiny-bootstrap-elements dir to your WordPress plugins dir
    Activate in WordPress admin plugins section
    Go to Tiny Bootstrap Elements Settings to configure your options
    Edit any page using your new Bootstrap toolbar
